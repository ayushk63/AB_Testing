# A/B Testing

### Aim 

Performed A/B Testing on Customer Satisfaction Scores and Behavior Data. 

## Chi-Squared Test

### Null Hypothesis: 
Gender and Purchase History are independent.

### Alternative Hypothesis:
Gender and Purchase History are dependent.

### Conclusion:
Failed to Reject Null Hypothesis. Therefore, Gender and Purchase History are independent.

## Independent Two-Sampled T-Test

### Null Hypothesis: 
There is no significant difference between the average satisfaction scores of Groups A and B.

### Alternative Hypothesis:
There is a significant difference between the average satisfaction scores of Groups A and B.

### Conclusion:
Failed to Reject Null Hypothesis. Therefore, there is no significant difference between the average satisfaction scores of Groups A and B.
